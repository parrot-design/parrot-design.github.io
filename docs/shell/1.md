# 常用Shell命令

### 目录操作

1. ls -- 查看文件/目录

常用：ls -l查看详细文件列表

2. pwd -- 查看当前目录

3. mkdir -- 创建目录

常用：mkdir 123 创建一个空文件夹

4. rm -- 删除文件/目录

常用：rm -rf 123 删除目录文件夹，并且目录下的文件一并删除
 
备注：直接使用命令【rm】只能删除空目录

-r 删除目录 -f强制删除文件或目录

5. mv -- 移动文件/目录改名、移动：

移动文件： mv 1.sh ./123 目标文件移动至某目录

6. unzip -- 解压zip文件格式

unzip [name].zip 解压到当前目录
