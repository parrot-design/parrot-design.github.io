# 考点一：vue 的相关概念

### 1.什么是 MVVM?

MVVM 是 `Model-View-ViewModel` 的缩写，`Model`代表数据模型，定义数据操作的业务逻辑，`View` 代表 UI 组件，它负责将数据模型转化成 UI 展示出来，`ViewModel` 通过双向绑定把 `View `和 `Model`进行同步交互，不需要`操作 DOM` 的一种设计思想。

### 2.请你说说对”渐进式框架“的理解

1. 所谓渐进式，其实就是 vue 的使用方式，同时也体现了 vue 设计的理念，渐进式代表的含义是：`主张最少`。
2. 我们在使用过程中，可以通过添加`组件系统`、客户端路由 `vue-router`、大规模状态管理 `vuex` 来构建一个完成的框架。
3. 更重要的是，这些功能相互独立，你可以在核心功能的基础上任意选用其他的组件，不一定要全部整合在一起。
