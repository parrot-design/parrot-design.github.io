# 考点五：CSS3 新特性

# 1.请说出你使用过哪些 CSS3 新特性？

  1. border-radius:用于实现圆角。
  2. box-shadow:用于实现阴影。
  3. border-image:用于实现边框图片。
  4. text-shadow:用于实现文字阴影。
  5. liner-gradient:用于实现背景线性渐变。
  6. transform:用于实现元素变形，包括旋转 rotate、扭曲 skew、缩放 scale 和移动 translate 以及矩形变形 matrix。
  7. transition:用于在一定的时间区间内，把元素从一种状态平滑地过渡到另一种状态。
  8. animation:结合@keyframes 创建实现动画。

# 2.请说出 CSS3 新增伪类有哪些？分别代表什么含义？

  1. p:first-of-type：选择属于其父元素的首个元素。
  2. p:last-of-type：选择属于其父元素的最后元素。
  3. p:only-of-type：选择属于其父元素的唯一的元素。
  4. p:only-child：选择属于其父元素的唯一子元素。
  5. p:nth-child(2)：选择属于其父元素的第二个子元素。
  6. :enabled :disabled 表单控件的禁用状态
  7. :checked 单选框或复选框被选中。
