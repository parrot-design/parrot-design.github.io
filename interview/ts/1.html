# TypeScript是什么？

TypeScript，简称 ts，是微软开发的一种静态的编程语言，它是 JavaScript 的超集。 那么它有什么特别之处呢?

  1. 简单来说，js 有的 ts 都有，所有js 代码都可以在 ts 里面运行。
  2. ts 支持类型支持，ts = type +JavaScript。
